# Expense Tracker Backend

A robust backend system for tracking expenses, built with **Spring Boot, JPA, MySQL**, and fully documented APIs using **Swagger UI**.  
This project provides analytics, CRUD operations, and export functionality for personal or business expense management.

---

## **Features**

- **User & Expense Management**: Create, read, update, and delete users and expenses  
- **Analytics**: Monthly, yearly, and category-wise expense summaries with totals  
- **Export Reports**: Generate PDF and Excel reports using **iText** and **Apache POI**  
- **API Documentation**: Fully documented and testable APIs via **Swagger UI**  

---

## **Tech Stack**

- **Backend:** Java, Spring Boot, JPA/Hibernate  
- **Database:** MySQL  
- **Documentation & Testing:** Swagger UI, Postman  
- **Reporting:** iText (PDF), Apache POI (Excel)  
- **Build Tool:** Maven
